Welcome to Project X.

Project X is the codename for the first project of the Hacklab of the Lawrence Creates Makerspace. You can find more about the MakerSpace here http://lawrencecreates.info/

We are planning on creating a quadcoptor project together as a team. Everyone is welcomed to join.

Team members, You can update this website yourself. Just sign up on github and you can create your own version of the website and send it to the team and we will then review it and take your changes. You will then be added to the team as well when we know your user name. You can contact @h4ck4rm1k3 if you have any questions.

Here are the instructions for forking the project :

$ git clone git@github.com:lawrencecreates/projectX.git
$ cd projectX
$ git fetch origin
$ git checkout gh-pages
